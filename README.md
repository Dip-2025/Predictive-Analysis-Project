# Boston Housing Dataset Analysis

## Overview
This project analyzes the Boston Housing dataset using Python.  
The dataset contains information about housing values in suburbs of Boston
and is commonly used for regression and exploratory data analysis.

## Dataset Description
- **Number of observations:** 506  
- **Number of features:** 13 predictor variables + 1 target variable  
- **Target variable:** Median value of owner-occupied homes (MEDV)

Each row represents a suburb in the Boston area, and each column represents
a socio-economic or environmental characteristic such as crime rate,
number of rooms, or distance to employment centers.
